UnRAG is a **RAG installer** for TypeScript projects. Instead of shipping a framework, it installs a **tiny, composable module** (ingest + retrieve + adapters) into *your* repo as source files you own.

## The 30-second mental model

- **Ingest**: chunk → embed → store (Postgres + pgvector)
- **Retrieve**: embed query → vector similarity search → return chunks

## Quick start

```bash
bunx unrag init --yes --store drizzle --dir lib/unrag --alias @unrag
```

Then (server-only):

```ts
import { createUnragEngine } from "@unrag/config";

const engine = createUnragEngine();
await engine.ingest({ sourceId: "docs:intro", content: "Hello from UnRAG" });
const result = await engine.retrieve({ query: "hello", topK: 5 });
```
