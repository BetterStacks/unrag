## What you’ll learn

- What RAG is (and is not) in practical, production terms
- The end-to-end system shape: ingestion → retrieval → context building → generation
- How to think about the core tradeoffs (quality vs latency vs cost)

## Prerequisites

None. If you can read API pseudocode, you can follow along.
